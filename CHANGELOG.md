# Changelog

## v2.0.0beta (2026-06-07) - release

First release based on the new `md-framebuffer-template` framework, with a rewritten demo set and menu system focused on classic Atari and demoscene effects such as animated menus, parallax scrolling, filled 3D vectors, multi-sprite stress tests, and Cojorotozoom-style intros. This release also improves sprite/tile rendering, main-loop performance, flash/RAM asset handling, clocking and DMA behavior, debug launch setup, and support for Atari ST, STE, Mega ST, and Mega STE.

## v1.0.3beta (2026-01-23) - release
- Bumped version and Atari ST firmware version.
- Optimized sprite/tile rendering and main loop performance.
- Moved sprite/tile/font assets to flash with RAM copies at runtime.
- Tweaked clocking and DMA priorities/behavior.
- Removed unused ROM_TEMP section, local settings, and unused libraries (sdcard, lwipopts, fatfs).
- Added AGENTS.md and improved debug launch behavior.
